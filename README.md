### 설치하는 방법
- 이 레포지토리를 다운받습니다.
- 파이썬을 설치합니다(3.6.x 버전을 설치하세요)
- 파이썬 discord 모듈을 설치합니다.
- cmd에 아래 명령을 입력해서 discord 모듈을 설치하세요:
```
pip install discord
```
- 돈받기 기능을 위해 splite3 db를 사용합니다. 기본 db파일이 레포지토리에 들어있습니다.
- `bot.py` 파일을 실행합니다.
